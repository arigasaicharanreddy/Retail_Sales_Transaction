## 📌 Project Overview
This project is an **End-to-End Data Science and Machine Learning pipeline** designed to analyze retail transactions, predict sales revenue, and identify high-value purchases. 

It takes raw transaction data, processes it into a relational database (MySQL), trains machine learning models to extract business intelligence, and deploys these models as a production-ready RESTful web service using **FastAPI**.

**Key Objective:** To empower retail businesses with real-time transaction predictions and actionable sales insights, bridging the gap between raw data and deployed software.

## 🚀 Key Features & Business Impact
- **Automated Data Pipeline:** Cleanses and loads raw CSV data into a MySQL database using SQLAlchemy, ensuring SQL compatibility.
- **Revenue Prediction (Regression):** Utilizes a **Gradient Boosting Regressor** to accurately predict the total transaction amount based on item price, quantity, discount, category, and payment method.
- **Customer Segmentation (Classification):** Uses a **Random Forest Classifier** to categorize transactions as "High Value" or "Standard Value," enabling targeted marketing strategies.
- **Interactive Web API:** Exposes the machine learning models via a fully documented **FastAPI** interface, allowing seamless integration with front-end web applications or POS systems.
- **Data Visualization & Reporting:** Automatically generates visual business dashboards (`Seaborn`/`Matplotlib`) and exports summary reports to Excel (`pandas`).

## 🌐 API Endpoints

The project features a built-in Swagger UI for easy testing. Once the server is running, navigate to `/docs` in your browser.

| Method | Endpoint | Description |
| --- | --- | --- |
| `GET` | `/` | API Health Check and Welcome Message. |
| `POST` | `/predict` | Accepts transaction details (JSON) and returns the predicted `Total Amount` in USD. |

**Sample API Request Body:**

{
  "Quantity": 3,
  "Price": 50.0,
  "DiscountApplied_Percent": 5.0,
  "ProductCategory": "Electronics",
  "PaymentMethod": "Credit Card"
}

## 💻 How to Run Locally

### 1. Clone the repository

```bash
git clone [https://github.com/arigasaicharanreddy/Retail_Sales_Transaction.git](https://github.com/arigasaicharanreddy/Retail_Sales_Transaction.git)
cd Retail_Sales_Transaction

```

### 2. Install Dependencies

```bash
pip install fastapi uvicorn pandas scikit-learn sqlalchemy pymysql matplotlib seaborn openpyxl joblib

```

### 3. Database Setup (Optional for API)

Make sure you have MySQL installed and running locally. Update the database credentials (`db_user`, `db_password`) in `main.py` if you wish to run the training script from scratch.

### 4. Run the Machine Learning Pipeline (Training)

*Note: The pre-trained `.joblib` models are already included in the repo, so you can skip this step if you just want to test the FastAPI server!*

```bash
python main.py

```

### 5. Start the FastAPI Server

uvicorn fastapii:app --reload



Once running, open your browser and go to **`http://127.0.0.1:8000/docs`** to test the machine learning predictions!
